PHP

Basic Information

Created in 1994 by Danish-Canadian programmer Rasmus Lerdorf

Originally used to track visits to users online resume, written in C language.
Was named "Personal Home Page Tools", AKA PHP Tools. The program evolved over the years,
eventually becoming a programming language in its own right. Today, PHP is used for website
development and server side scripting, and is used by Facebook, Wikipedia, and other notable websites.
