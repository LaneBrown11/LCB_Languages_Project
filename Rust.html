Rust

Basic Information

Created by Graydon Hoare in 2006, released officially in 2010

Rust is a relatively new language that focuses on reliability and safety. It's known to
provide strong memory safety, good performance, and low level control. It can be compared
to C and C++, providing low level control, but unlike these older languages, Rust is far less
prone to crashes, bugs, and security threats. It is commonly used to build kernels, drivers, and other
components where memory is a crucial aspect. Large social companies like Facebook, Discord, and Amazon
use Rust to improve security in their systems.
Compared to older languages, Rust is

  • Faster
  • More Stable
  • Emphasizes safety
