C#

Basic Information

Created in 2000 by Microsoft led by Anders Hejlsburg

The original goal of hejlsburg and team was to make C-like object oriented language. Upon release,
Java creator James Gosling claimed it was a less feature complete copy of java. Over time however,
the two languages deviated more in functionality, with C# becoming well adapted for use on windows
operating systems. C# is still used today by microsoft and the windows system, but also by

  1. Unity Engine
  2. Electronic Arts
  3. Intel
  4. Amazon
